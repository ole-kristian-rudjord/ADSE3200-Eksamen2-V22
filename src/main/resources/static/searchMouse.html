    Search Mouse page

    The "Search Mouse" page displays a table filled with mice from ES' current database. In the top left corner, is a button which allows for custom filters to be applied. This will let you search for mice that fit within certain specifications like size, weight, connectivity, etc.

    Mouse specifications:

    Brand

    refers to mice's producer or the company which sells the product.

    Model

    specifies the name of specific mouse from a brand.

    Length

    Width

    Height

    Weight

    is measured in grams, preferably without the cable:

    Shape

    Connectivity

    refers to whether a mouse is wired or wireless.

    Sensor

    specifies the sensor-version that is used in a mouse.

    DPI

    describes to the maximum Dots Per Inch (DPI) available for a mouse.
    Learn more about DPI here.

    Polling rate

    refers the maximum amount of times a mouse can send information per second.
